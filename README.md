# Micro Tool Network

A static directory site for organizing a network of free browser tools and low-cost Gumroad kits.

## Strategy

- Build free privacy-first tools.
- Group tools into clusters.
- Link every tool to a hub, sibling tools, and a paid kit.
- Use Gumroad for template/checklist/toolkit monetization.
- Submit each hub to Google Search Console with a sitemap.
